# ECHOMEN Ecosystem Website



The official website for the ECHOMEN Ecosystem - The Autonomous Agent System Built for Action.

## Overview

ECHOMEN is a dual-core ecosystem that seamlessly switches between a high-speed CLI (`echoctl`) and a powerful Web UI. Execute tasks, manage MCP servers, and automate your workflow with precision and speed.

- **Frontend:** React 19 + Vite
- **Styling:** Tailwind CSS 4 + Shadcn/UI
- **Animations:** Framer Motion
- **Deployment:** Manus + GitHub Pages
- **Hosting:** Multi-cloud infrastructure

```bash
# Production build
pnpm build

# Preview production build
pnpm preview
```
